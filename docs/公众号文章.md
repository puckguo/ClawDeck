# 我花了一个月，做了个省心的 AI Agent 管理工具

> 从命令行到可视化，从配置地狱到一键搞定

![ClawDeck 主界面](docs/images/agent1.png)

---

## 01 为什么是 ClawDeck？

作为一个重度 AI 工具使用者，我一直在用 OpenClaw 框架部署各种 Agent。但说实话，**配置文件写多了真的头大**。

每次要改个配置，得记住：
- `openclaw.json` 放在哪儿？
- SOUL.md 和 IDENTITY.md 是什么格式？
- 飞书的 AppID 填在哪儿？
- 这个 Agent 用的哪个 AI 模型？

更崩溃的是，**配置错了没有提示**，Agent 启动失败也不知道问题出在哪。

有一天我盯着满屏的配置文件想：**为什么不能像管理服务器一样，有个可视化的界面？**

于是 ClawDeck 诞生了。

---

## 02 开发的心路历程

### 第一周：需求纠结

一开始只是想做个简单的配置编辑器。但越想越复杂：
- 要不要支持多 Agent 管理？
- 配置文件改了怎么备份？
- 飞书配置太麻烦，能不能简化？

最后决定：**做就做个完整的，能真正解决问题的工具。**

### 第二周：技术选型

选了 React + Node.js 的技术栈，原因很简单：
- 前端用 Ant Design，组件丰富，快速搭建
- 后端用 TypeScript，类型安全，维护方便
- 前后端分离，以后可以独立迭代

### 第三周：踩坑实录

**坑 1：配置格式不统一**

OpenClaw 的配置文件分散在各处，有的在工作空间，有的在 Agent 目录。花了两天时间才理清楚文件关联关系。

**坑 2：飞书集成调试难**

飞书的 WebSocket 连接经常断，消息收不到。后来加了自动重连机制，才算稳定。

**坑 3：Windows 兼容性**

开发在 Mac 上很顺利，到 Windows 上一测试，路径分隔符、进程管理、CPU 监控全都有问题。又花了一周做跨平台适配。

### 第四周：打磨体验

加了一个**使用向导**功能，第一次打开时会引导用户了解各个功能。还加了**自动备份**，每次修改配置文件前都会存一份到 `.backups` 目录。

## 03 功能一览

### 🎯 Agent 列表 - 一目了然

![Agent 列表](docs/images/agent1.png)

所有 Agent 的状态一目了然：
- 🟢 运行中
- 🔴 已停止
- 🟡 配置中

点击卡片就能进入详情页，不用再记配置文件路径。

---

### 🚀 创建向导 - 四步搞定

![创建向导](docs/images/create-agent.png)

以前创建一个 Agent 要：
1. 手动创建目录
2. 写 openclaw.json
3. 配置 AI 模型
4. 设置飞书参数
5. 启动服务

**现在四步搞定**：基本信息 → AI 配置 → 渠道设置 → 确认创建。还支持一键启动！

---

### 📊 概览面板 - 实时监控

![概览面板](docs/images/概览1.png)

最爽的功能之一。可以看到：
- 实时 CPU / 内存占用
- 运行时长
- 已安装的技能
- 当前加入的房间

数据每 5 秒自动刷新，像看服务器监控一样直观。

---

### 📝 配置管理 - 可视化编辑

![配置管理](docs/images/config.png)

再也不用记配置文件格式了：
- AI 模型下拉选择
- 参数表单填写
- 实时验证提示

支持 DeepSeek、Kimi、OpenAI 等多种模型，选错了会有提示。

---

### 💬 飞书集成 - 一键配置

![飞书配置](docs/images/配飞书1.png)

飞书机器人配置曾经是最头疼的部分。现在在界面上填好 AppID 和 AppSecret，**自动完成配置**。

支持：
- WebSocket 长连接
- 私聊 / 群聊
- 白名单控制
- 自动重连

---

### 🎭 身份设定 - 可视化编辑

![身份设定](docs/images/身份1.png)

SOUL.md 和 IDENTITY.md 是 Agent 的"灵魂"文件。现在可以直接在网页上编辑，**自动保存备份**。

左边写配置，右边看效果，支持 Markdown 预览。

---

### ✏️ Markdown 编辑器 - 所见即所得

![Markdown 编辑器](docs/images/md1.png)

内置了功能完善的 Markdown 编辑器：
- 语法高亮
- 实时预览
- 自动保存
- 历史版本

写技能说明书、性格设定都特别方便。

## 04 技术亮点

虽然是个管理工具，但技术上也有些小心思：

**🔄 自动备份机制**

每次修改配置文件前，自动创建备份到 `.backups` 目录，按时间命名。改错了可以一键恢复。

**📱 响应式设计**

支持桌面端和移动端访问，出门在外也能查看 Agent 状态。

**🌐 中英双语**

支持中文和英文切换，国际化界面。

**⚡ 实时监控**

后端用 WebSocket 推送状态变更，前端即时更新，不用手动刷新。
